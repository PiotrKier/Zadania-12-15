Wyliczenia


Możliwości formatowania tekstu w HTML.


  • Indeksgórny: < sup > < /sup >
  • Indeksdolny: < sub > < /sub >
  • Pogrubienie: < b > < /b >

  • Kursywa: < i > < /i >
  • Podkreślenie: < u > < /u >
  • Przekreślenie: < strike > < /strike >

  • Nagłówek dokumentu: < head > < /head >
  • Właściwa treść dokumentu: < body > < /body >
  • Tytuł strony: < title > < /title >



Numerowanie


Możliwości funkcji FONT.

  1. < font size ="Wielkość czcionki">
  2. < font color ="Kolor czcionki">
  3. < font face ="Rodzaj czcionki">

Możliwości funkcji ALIGN.

  1. < Align size ="Wielkość tekstu">
  2. < Align color ="Kolor tekstu">
  3. < Align ="center"> = Wyśrodkowanie tekstu

Nagłówki w treści strony WWW.
